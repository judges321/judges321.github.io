Foundation框架

Foundation框架

  1. Class类
  2. Functions函数
  3. Constants:Foundation Framework中定义的常量，such as NSError Codes

1.1 Data Storage

NSData和NSString为字节数组，NSValue和NSNumber为数值数组，NSArray、NSObject和NSSet为对象

1.2 Text and String
NSCharacter代表多种NSString和NSScanner使用的字符分组，NSString提供代表文字的字符串，并提供搜索，合并，比较等操作方法，NSScanner用来在NSString对象中查找数组单词

1.3 data and times
NSFormatter的用途是“将数据在字符串与特定类型的对象之间转换”

1.4 Application coordination and timing(应用程序协作及定时器)

UI框架MFC、QT、WTL或者WPF最骨架部分为消息管理系统，称之为消息泵（Message Pump),NSTimer、NSNotificationCenter模拟消息泵

1.5 Operating-system services(操作系统底层服务)

NSFileManager用于提供一个文件操作（增删改）的一致借口，NSThread和NSProcessInfo是我们能够创建多线程应用和请求应用运行时环境。耗时的工作放在后台工作线程里完成，放主线程里会卡死。iOS线程技术NSThread和NSOperation和GCD（Grand Center Dispatch）

1.6 URL Loading System（网络功能）

AFNetworking第三方库，网络功能的第三次封装
